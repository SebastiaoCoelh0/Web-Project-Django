Base URL

Endpoints Disponíveis

Listar Cidades

Método: GET

URL: api/cidades/

Descrição: Retorna uma lista de todas as cidades disponíveis.

Previsão de Hoje para uma Cidade

Método: GET

URL: api/previsao_hoje/<int:cidade_id>/

Descrição: Retorna a previsão do tempo para hoje para a cidade especificada.

Parâmetros:

  • cidade_id (integer): ID da cidade.

Resposta: JSON

{
    "cidade": "Lisboa",
    "data": "2023-07-23",
    "min_temp": 18,
    "max_temp": 27,
    "descricaoTempo": "Céu limpo",
    "precipitacao": 0,
    "icon_url": "https://www.ipma.pt/bin/images/weather-symbols/w_ic_d_01anim.svg"
}

Previsão dos Próximos 5 Dias para uma Cidade

Método: GET

URL: api/previsao_proximos_5_dias/<int:cidade_id>/

Descrição: Retorna a previsão do tempo para os próximos 5 dias para a cidade especificada.

Parâmetros:

  • cidade_id (integer): ID da cidade.

Resposta: JSON

    "cidade": "Lisboa",
    "previsoes": [
